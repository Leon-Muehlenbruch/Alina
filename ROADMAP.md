## Geplant

### QR-Code zum Kontakte hinzufügen
Statt einen 6-stelligen Code abzutippen einfach QR-Code zeigen — die andere Person scannt ihn direkt mit der Kamera. Ideal für Treffen vor Ort. Macht den Einladungsflow deutlich schneller und intuitiver.

### Sprachnachrichten
Mikrofon-Button im Chat — drücken, sprechen, loslassen, senden. Besonders hilfreich für alle, die lieber sprechen als tippen.

### Selbstlöschende Nachrichten
Ablaufzeit pro Chat oder pro Nachricht festlegen (z.B. 24 Stunden, 7 Tage). Nach Ablauf werden Nachrichten automatisch aus dem lokalen Speicher entfernt. Passt zum Datenschutz-Fokus von Alina.

### App-Sperre mit PIN
Beim Öffnen der App einen PIN eingeben — niemand kann einfach auf fremdem Handy mitlesen. Optional mit Fingerabdruck/Face ID auf unterstützten Geräten.
